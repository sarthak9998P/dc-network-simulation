# Data Center Network Simulation Lab

## Overview
Simulated data center network using Cisco Packet Tracer.

## Topology
- Router: Cisco 1941
- Switch: Cisco 2960-24TT  
- 3 PCs configured with static IPs

## Network Details
- Network: 192.168.1.0/24
- Router IP: 192.168.1.1
- PC0: 192.168.1.10
- PC1: 192.168.1.11
- PC2: 192.168.1.12

## Skills Demonstrated
- Network topology design
- Router & Switch configuration
- IP addressing & subnetting
- Connectivity testing (ping)
- Data center network simulation
